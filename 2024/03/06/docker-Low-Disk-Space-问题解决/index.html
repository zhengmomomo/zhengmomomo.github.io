docker Low Disk Space 问题解决

docker 编译镜像时Low Disk Space On “Filesystem root” The volume “Filesystem root“ has only 0 disk space remaining

docker info查看当前docker存储路径，默认为/var/lib/docker

docker system df 查看docker空间使用情况

df -h 查看本机磁盘使用情况，找到空间剩余较大的磁盘，更改docker存储位置

编辑 /etc/docker/daemon.json文件

添加

“data-root”: “/xxx”,

xxx为docker的新存储位置，修改后保存退出

配置生效并重启docker服务
